Instructions to run the code:

Follow instructions on: https://github.com/carlosjimenezmwb/seq_nms_yolo, with the following exceptions:

On step 2, run:

source activate EnvExample, instead of: conda activate EnvExample.

Skip step 7 since the video is already in the folder. If you want to select a different video, then change the name of the video. This code provides 5 sample .avi files: input.avi, input2.avi, input3.avi, input4.avi, and input 5.avi. This files correspond, respectively, with the following files in the UCF-101 Dataset: v_ApplyEyeMakeup_g19_c03.avi, v_ApplyLipstick_g21_c01.avi, v_Archery_g07_c01.avi, v_BabyCrawling_g11_c01.avi, v_BalanceBeam_g18_c01.avi‚Äù.

On step 8, run (If the video is an .avi file):

python video2img.py -i input1.avi, instead of: python video2img.py -i input.mp4. If the video is a .mp4 file, then skip this change (name of file needs to be input.mp4, in this case). If the video is input2.avi, then run: python video2img.py -i input2.avi, if the video is input3.avi, then run: python video2img.py -i input3.avi, if the video is input4.avi, then run: python video2img.py -i input4.avi, and finally, if the video is input5.avi, then run: python video2img.py -i input5.avi.

Yolo without seq_nms:

To remove seq_nms from the tracking algorithm, the following lines of code are removed/commented in yolo_seqnms.py (suggested by https://github.com/hectordavideps/seq_nms_yolo):

-maxPath(dets,links) line 205

-NMS(dets) line 206

Then, repeat steps 8 to 11 of instructions on first Github, to get results without seq_nms.
